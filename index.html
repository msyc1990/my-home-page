Kilka słów o sobie

Jestem technikiem informatykiem, który obecnie uczy się programowania na stanowisko Frontend Developera. Posiadam podstawową wiedzę w takich technologiach jak HTML5, CSS, JavaScript oraz Responsive Web Design. Chcę zdobyć doświadczenie w branży oraz dalej rozwijać swoje umiejętności.

W moim portfolio znajdziesz kilka projektów, na których pracowałem, które obejmują wiele różnych technologii webowych. W szczególności specjalizuję się w tworzeniu interaktywnych i responsywnych interfejsów użytkownika, korzystając z HTML5, CSS, JavaScript

Znajomość technologii

JavaScript

  • znajomość podstawowych typów danych (np. liczby, łańcuchy znaków, obiekty)
  • umiejętność tworzenia i manipulowania zmiennymi i obiektami
  • znajomość struktur kontroli przepływu programu, takich jak instrukcje warunkowe i pętle
  • umiejętność tworzenia funkcji i wykorzystywania ich w programie

HTML-5

  • znajomość podstawowych elementów HTML, takich jak nagłówki, paragrafy, listy i tabele
  • umiejętność korzystania z formularzy i elementów interaktywnych, takich jak przyciski i linki
  • znajomość semantycznego znaczenia elementów HTML i umiejętność stosowania ich w celu poprawy dostępności i SEO
  • umiejętność korzystania z atrybutów, takich jak "class" i "id", do stylizacji i manipulacji elementami HTML
  • znajomość mechanizmów wstawiania multimediów, takich jak obrazy, audio i wideo, na stronie

CSS

  • znajomość selektorów CSS i umiejętność ich stosowania w celu wybierania elementów HTML
  • umiejętność stosowania różnych typów stylów CSS, takich jak kolor, rozmiar czcionki, marginesy i obramowania
  • znajomość modelu pudełkowego CSS i umiejętność stosowania go w celu precyzyjnej kontroli układu strony
  • umiejętność stosowania kaskadowych stylów CSS, takich jak styl strony (style sheet) i styl wewnętrzny (inline style)
  • znajomość technik responsywnego projektowania stron, takich jak media queries i flexbox, w celu dostosowywania strony do różnych rozmiarów ekranu
